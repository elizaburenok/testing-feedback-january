## 3. Отступы (Spacing)

Система отступов основана на базовом значении 4px:

- `Spacing 1x`: `4px`
- `Spacing 6x`: `24px`
- `Spacing 8x`: `32px`
- `Spacing 12x`: `48px`

**Паттерн**: Базовое значение умножается на множитель (1x, 6x, 8x, 12x)

## 5. Структура именования

### Формат именования токенов:
```
[Категория]/[Подкатегория]/[Вариант] [Модификатор]
```

### Примеры:
- `Primitive/Primary` - примитивный основной цвет
- `Primitive/Primary Fixed` - фиксированная версия (не меняется при смене темы)
- `Primitive/Primary On Inverse` - основной цвет на инверсном фоне
- `Bg/Neutral 1` - фоновый нейтральный цвет уровня 1
- `Container/Transparent 1` - прозрачный контейнер уровня 1

---

## 6. Особенности системы

### 6.1. Fixed варианты
Многие токены имеют версию "Fixed", которая не изменяется при переключении темы (например, `Primary Fixed`, `Secondary Fixed`).

### 6.2. On Inverse варианты
Цвета, предназначенные для использования на инверсном (темном) фоне, имеют суффикс "On Inverse".

### 6.3. Альфа-каналы
Система использует цвета с альфа-каналом для прозрачных элементов:
- `Alpha/Neutral 020/005` - 5% непрозрачности
- `Alpha/Neutral 020/040` - 40% непрозрачности
- `Alpha/Neutral 020/050` - 50% непрозрачности

### 6.4. Иерархия нейтральных цветов
Нейтральные цвета организованы по уровням от 1 до 5, где:
- **Neutral 1** - самый светлый
- **Neutral 5** - самый темный

### 6.5. Брендовые цвета
Брендовые цвета имеют градацию от 1 до 5, где:
- **Brand 1** - самый светлый оттенок
- **Brand 5** - более насыщенный оттенок
- **Brand** (без номера) - основной брендовый цвет

## 8. Рекомендации по использованию

1. **Примитивные цвета** (`Primitive/*`) следует использовать для текста и основных элементов интерфейса
2. **Фоновые цвета** (`Bg/*`) предназначены для фоновых элементов
3. **Контейнеры** (`Container/*`) используются для контейнеров с различной прозрачностью
4. **Страницы** (`Page/*`) определяют основной и вторичный фоны страниц
5. **Наложения** (`Overlay/*`) используются для модальных окон и затемнений
